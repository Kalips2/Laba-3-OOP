                Можливості програми:
1. Дивитись на структуру файлів вашого C and D дисків.
2. Відкривати, копіювати, вставляти, видаляти файли з папок.
3. У конкретній папці шукати файл, назва якого містить вираз, який ввів користувач. (Завдання розділу 0.1)
4. Для відкриття файлів реалізован патерн AbstractFactory (Завдання розділу 2)
    (Зауваження: Маємо два абстрактні класи для різної роботи: FileHelper для файлів, окрім html, HtmlHelper для файлів типу html
    OpenTxt, OpenXml, OpenAnotherFile й відповідні фабрики містять майже однакову реалізацію, бо завдання варіанту не
    вимогало чогось різного для відкриття. Але я їх добавив як окремі, щоб можно було наглядно побачити реалізацію патерна.)
5. Під час відкриття файлу html є доступний функціонал: (Завдання розділу 1)
    1) Заміна тегу на новий
    2) Перегляд усіх тегів файлу
    3) Перегляд коду файлу. (І у браузері, і у як код).
